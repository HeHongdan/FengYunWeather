# FengYunWeather 风云天气

**FengYunWeather** 是一个开源的android 天气app，使用kotlin语言编写，采用MVVM架构，使用了kotlin协程，OkHttp，coil，Room等框架，代码逻辑非常清晰，适合新手入门，以及老手参考交流。



## 主要功能

- 全球每日和逐小时天气预报，包括：天气状况、温度、大气压、相对湿度、降水量、风力风速风向、能见度、日出日落和月升月落等
- 中国城市还有AQI空气质量数据
- 逐小时天气动态曲线
- 城市搜索，支持模糊搜索和搜索建议
- 多语言，目前支持中英文
- 单位转换
- 可设置不同字体大小
